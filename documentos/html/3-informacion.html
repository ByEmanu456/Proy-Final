Límites

Texto a voz

¿Qué son los límites?

Las funciones matemáticas se utilizan en otros ámbitos, por ejemplo, para calcular los beneficios o los costes de una empresa, la velocidad o aceleración de un móvil, entre muchas otras cosas, por lo que es importante conocer el comportamiento de una función.

El límite de una función es el valor al que f(x) se acerca a medida que x se aproxima a algún número. Por decirlo de otra manera, los límites describen cómo se comporta una función cerca de un punto, en vez de en ese punto. Esta simple pero poderosa idea es la base de todo el cálculo.

¿Cómo evaluar límites?

Encontrar límites para la gran mayoría de puntos para una función dada es tan simple como sustituir el número que x se acerca en la función. Dado que esto convierte la evaluación de límites en una sustitución a nivel álgebra, la mayoría de las preguntas que involucran límites se centran en los casos en los que la sustitución no funciona.

Sustitución

Encontrar un límite directamente significa evaluar la función sustituyendo el valor al que x se acerca en la función y evaluar el resultado. Este método garantiza encontrar el resultado siempre y cuando no se termine dividiendo entre 0.

Indeterminaciones

Una indeterminación ocurre cuando es imposible evaluar el límite de forma directa porque nos encontramos con una división entre 0. Para estos casos es necesario usar algunos "trucos matemáticos" como:

  • Factorizar el numerador y el denominador para ver si se puede eliminar la indeterminación.
  • Racionalizar mediande conjugación las raices en caso de que estas ocacionen la indeterminación.
  • Utilizar productos notables para intentar eliminar los terminos que provocan la indeterminación.
  • Probar utilizando indentidades trigonométricas en caso de que las haya.
  • Hacer aproximación. Esta es la ultima opción puesto que se utilizan tablas y gráficas para aproximar el valor del limite.

En caso de que nada funcione, se puede decir que el límite es una verdadera indeterminación ya que puede ser cualquier cosa. Puede ser 0, ∞, 1, 14, puede ser cualquier número.
