Funciones Matemáticas

Texto a voz

¿Qué son las funciones?

Una función matemática es una relación que se establece entre dos conjuntos, a través de la cual a cada elemento del primer conjunto se le asigna un único elemento del segundo conjunto o ninguno.

Al conjunto inicial o conjunto de partida también se lo llama dominio; al conjunto final o conjunto de llegada, en tanto, se lo puede denominar contradominio, rango o imagen.

Al elemento genérico del dominio se lo conoce como variable independiente; al elemento genérico del contradominio, como variable dependiente.

Como tal, se puede aplicar a diversas situaciones, tanto en la vida cotidiana como en las ciencias, donde se advierten relaciones de dependencia entre dos elementos.

Clases de funciones matemáticas

Las funciones se puedes clasificar de la siguiente manera:

Algebraicas

  • Funciones lineales
  • Funciones cuadráticas
  • Funciones cúbicas
  • ...

Trascendentales

  • Funciones trigonométricas
  • Funciones exponenciales
  • Funciones logarítmicas

Funciones lineales

Llamamos función lineal a las funciones polinómicas de primer grado, es decir, una función cuya representación es una línea recta. Su expresión general es f ( x ) = mx + n.

Funciones cuadráticas

Es una función polinómica con una o más variables en la que el término de grado más alto es de segundo grado. Son de la forma f ( x) = ax² + bx + c, con a ≠ 0

Funciones cúbicas

Tambien se les llama funciones polinomial de grado 3. Puede ser escrita en la forma f ( x ) = ax 3 + bx 2 + cx + d

Funciones trigonométricas

Son funciones asociadas a una razón trigonométrica. Las 3 más comunes son: la función seno, coseno y tangente.

Funciones exponenciales

Se llaman funciones exponenciales a las que tienen la ecuación f ( x ) = abx, siendo la base “a” un número real positivo distinto de cero.

Funciones logarítmicas

Las funciones del tipo y = logax llamadas logarítmicas son la inversa de la función exponencial f ( x ) = ax. Siendo la base “a” un número real positivo distinto de cero.
